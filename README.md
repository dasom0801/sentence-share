# Sentence Share
책 속의 문장 공유 서비스

https://sentence-share.firebaseapp.com/

+ 기간: 2019. 03. 10 ~ 03. 24
+ 사용 기술
  + HTML
  + CSS(SCSS)
  + React
    + create-react-app
    + react-router
    + redux
    + redux-thunk
  + immutable.js
  + axios
  + firebase
  + 카카오 검색 API
  + 반응형 웹
+ 주요 기능
  + 회원가입 및 정보수정
    + 구글 계정으로 회원가입
    + 닉네임과 이미지 변경
  + 문장 공유 및 수정
    + 카카오 검색 API로 해당 문장의 출처 도서를 검색하고 등록
    + 검색결과가 없는 경우 사용자가 직접 입력
  + 최신순/인기순 정렬
  + 내가 등록한 문장, 좋아요를 한 문장보기
  + 책, 등록한 사용자별로 문장 보기
